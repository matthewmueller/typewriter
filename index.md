## Typewriter()

  Initialize `Typewriter`

## Typewriter.start()

  Start the typewriter
  
```js
typewriter.start()
```

## Typewriter.stop()

  Stop the typewriter
  
```js
typewriter.stop()
```

## Typewriter.restart()

  Restart the typewriter. Clears the input
  
```js
typewriter.restart()
```

## Typewriter.clear()

  Clear the typewriter input
  
```js
typewriter.clear();
```
